USB 2.0 Host Controller

Information

Type: USB 2.0 Host Controller
Vendor: GOWIN Semiconductor

Summary

Gowin USB 2.0 Host Controller IP complies with the Enhanced Host Controller Interface Specification(EHCI). Supports SRAM interface and can be connected to any microprocessor that supports SRAM interface. Supports ULPI interface and can be connected to any USB2.0 transceiver that supports ULPI interface. It contains up to 32KB RAM space to store TDs and data.
