AXI-Stream FIFO

Information

Type: AXI-Stream FIFO
Vendor: GOWIN Semiconductor

Summary

AXI-Stream FIFO converts memory mapped transactions to AXI-Stream transactions and vice versa. Choose desired data interface(AXI4 or AXI4-Lite), FIFO depth, Data width of AXI interface, and transmit mode.
