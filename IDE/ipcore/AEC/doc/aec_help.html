AEC

Information

Type: AEC
Vendor: GOWIN Semiconductor
Summary: Gowin AEC(Acoustic Echo Cancellation) IP is used to remove the echo from the captured signal. This IP can judge whether the captured signal contains echoes by XCORR, and use NLMS(Normalized Least Mean Square) adaptive filter to filter out the echo part with high correlation in the captured signal to realize echo cancellation.

Options & Description

Data Options

Data width :

  • Specify the data width.

XCORR Options

Number of points :

  • Specify the points of XCORR.

Max Lag :

  • Specify the value of maximun lag. If specify maxlag, the returned XCORR sequence ranges from -maxlag to maxlag.

Normalized Threshold :

  • Specify the normalized threshold of XCORR.

Threshold :

  • Specify the threshold of XCORR.

NLMS Options

Step Size :

  • Specify the adaptive step size in NLMS Adaptive Filter.

Number of Taps :

  • Specify the number of taps in NLMS Adaptive Filter.
