MIPI_CPHY

Information

Type: MIPI_CPHY
Vendor: GOWIN Semiconductor

Summary

The Gowin MIPI-CPHY supports combo RX/TX C-PHY for bidirectional data transmission. The MIPI CPHY is compliant with the MIPI C-PHY v1.2 protocol, achieving a maximum data transmission rate of 2.5GSPS per lane. The MIPI CPHY supports three independent data channels and has a relatively rich and free clock interface, which can meet various application needs.
