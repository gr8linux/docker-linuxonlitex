FP Add Sub

Information

Type: FP Add Sub
Vendor: GOWIN Semiconductor

Summary

The Gowin FP Add Sub IP supports floating-point addition or subtraction between two single-precision format inputs. The IP supports optional output ports such as zero, overflow, underflow, and nan.
