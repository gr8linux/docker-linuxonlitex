AHB Bus Arbiter

Information

Type: AHB Bus Arbiter
Vendor: GOWIN Semiconductor

Summary

Gowin AHB Bus Arbiter is a fully verified soft IP core which can arbitrate multiple masters to access a slave. There are 16 masters and a slave interfaces in AHB Bus Arbiter. 16 masters have a fixed priority, the arbiter master interface 0 has the highest priority, the arbiter master interface 15 has the lowest priority. The AHB bus bit width can be configured as 32 bit or 64 bit, the 32 bit width is the bus standard, the 64 bit width is an enhanced design to improve communication rate. 16 masters of AHB Bus Arbiter can be MCUs with AHB bus system, or FPGA logic fabrics conform to AHB protocol.
