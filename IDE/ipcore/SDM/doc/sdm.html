SDM

Information

Type: SDM
Vendor: GOWIN Semiconductor

Summary

The Gowin SDM IP is a Sigma-Delta modulator that use oversampling, noise shaping, digital filtering to achieve conversion of PCM to PDM signals, and it supports configurable input data width.

The design has these features:

  • support 5 level filters
  • Support 16,18,24-bits per pixel input
