IIR Filter

Information

Type: IIR Filter
Vendor: GOWIN Semiconductor

Summary

The Gowin IIR Filter IP is primarily used for high data throughput applications that require a sharp cut-off characteristic. IIR filters require less hardware than FIR filters and have faster response. The Gowin IP supports multi-channel and multiplier time division multiplexing function. Moreover, the IP has the function of reloading the coefficients during operation, which is flexible and convenient to use.
