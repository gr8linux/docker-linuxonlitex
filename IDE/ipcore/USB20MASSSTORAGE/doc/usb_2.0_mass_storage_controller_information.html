USB 2.0 Mass Storage

Information

Type: USB 2.0 Mass Storage
Vendor: GOWIN Semiconductor

Summary

UMS is a USB mass storage device ,which realizes data exchange with the storage medium by sending SCSI commands.
Gowin USB 2.0 Mass Stroage IP uses BOT (Bulk-only transport) transfer protocol for mass data transfer between host and USB device.
Gowin USB 2.0 Mass Stroage IP supports two Flash control modes: default user-defined and w25q128fv.
