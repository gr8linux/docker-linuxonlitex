SEU Handler

Information

Type: SEU Handler
Vendor: GOWIN Semiconductor

Summary

An SEU (Single Event Upsets) with sufficient energy changes the logic state of the configuration memory element (i.e. SRAM), which is the heart of FPGAs. The Gowin SEU Handler core within GW5A family adopts an SEU mitigation technique so as to protect the configuration memory from soft errors. This module enables the soft error detection and correction by Error correction code (ECC) and cyclic redundancy check (CRC) algorithm per configuration memory frame. Error injection is supported for functional verification and evaluation.
