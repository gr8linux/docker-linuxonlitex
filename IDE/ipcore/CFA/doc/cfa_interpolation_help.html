Color Filter Array Interpolation

Information

Type: Color Filter Array Interpolation
Vendor: GOWIN Semiconductor
Summary: The Gowin Color Filter Array Interpolation IP is a high performance and resource efficient implementation for converting Bayer sensor data to the RGB color domain.

The design has these features:

  • Support Bayer image sensor
  • Support resolutions from 8 × 8 up to 4320 × 7680
  • 5 × 5 interpolation filtering
  • Support 8,10,12,16-bits per pixel input and output

Options & Description

Maximum Number of Columns :

  • Input Image Columns.

Maximum Number of Rows :

  • Input Image Rows.

Input Data Width :

  • Input Image Bits.

Output Data Width :

  • Output Image Bits.

Bayer Phase :

  • Input Image Bayer Phase.
