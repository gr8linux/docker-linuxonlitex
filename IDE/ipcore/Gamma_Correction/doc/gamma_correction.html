Gamma Correction

Information

Type: Gamma Correction
Vendor: GOWIN Semiconductor

Summary

The Gowin Gamma Corretion IP is a high performance and resource efficient implementation for correcting image data to match the response of display devices.

The design has these features:

  • Support RGB image
  • Support resolutions from 8 × 8 up to 4320 × 7680
  • 3 gamma LUTs
  • Support 8,10,12-bits per pixel input and output
