HDLC

Information

Type: HDLC
Vendor: GOWIN Semiconductor

Summary

Gowin HDLC(High-Level Data Link Control) IP is a controller for HDLC with the basic frame format. HDLC is located at the link layer of OSI (Open System Interconnection) reference model published by the ISO (International Organization for Standardization).
