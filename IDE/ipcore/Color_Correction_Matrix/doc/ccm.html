Color Correction Matrix

Information

Type: Color Correction Matrix
Vendor: GOWIN Semiconductor

Summary

The Gowin Color Correction Matrix IP is a 3x3 configurable coefficient matrix multiplier with offset compensation. The IP can be used for color correction such as color cast or contrast in an image.

The design has these features:

  • Support RGB image
  • Support resolutions from 8 × 8 up to 4320 × 7680
  • 3 × 3 coefficient matrix and 1 × 3 offset matrix
  • Support 8,10,12,16-bits per pixel input and output
