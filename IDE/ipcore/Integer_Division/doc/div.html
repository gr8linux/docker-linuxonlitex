Integer Division

Information

Type: Integer Division
Vendor: GOWIN Semiconductor

Summary

The Gowin Integer Division IP accept a integer dividend and a integer divisor as inputs, and produce two integer outputs, a quotient and a remainder. This IP supports signed and unsigned data representation for the dividend and divisor.
