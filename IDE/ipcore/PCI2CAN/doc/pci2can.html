PCI to CAN

Information

Type: PCI to CAN
Vendor: GOWIN Semiconductor

Summary

The Gowin PCI to CAN IP realizes mutual communication between Gowin PCI Target IP and Gowin CAN IP. The user can configure all registers in Gowin CAN IP through Gowin PCI Target IP and Gowin PCI to CAN IP.
