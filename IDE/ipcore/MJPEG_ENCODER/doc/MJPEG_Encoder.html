MJPEG Encoder

Information

Type: MJPEG Encoder
Vendor: GOWIN Semiconductor

Summary

The Gowin MJPEG Encoder IP is a frame by frame compression encoder for dynamic images, which can convert images from RGB format into YCbCr format and compress and store them.

The design has these features:

  • Support RGB image
  • Support resolutions from 64 × 64 up to 1920 × 1080
  • Support 24 bits per pixel input and 8 bits output
