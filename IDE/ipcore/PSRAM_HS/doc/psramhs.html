PSRAM Memory Interface HS

Information

Type: PSRAM Memory Interface High Speed
Vendor: GOWIN Semiconductor

Summary

This PSRAM Memory Interface Controller is for interfacing devices with embedded Winbond PSRAM. This controller interfaces to a single 8-bit PSRAM memory. V1.0 has PLL delay auto calibration within the IP. V2.0 has a fixed delay value that is configured while generating the IP with a default value optimized for embedded devices.
