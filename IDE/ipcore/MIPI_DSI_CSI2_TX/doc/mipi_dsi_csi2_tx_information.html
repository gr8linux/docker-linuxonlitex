MIPI DSI/CSI-2 Transmitter

Information

Type: MIPI DSI/CSI-2 Transmitter
Vendor: GOWIN Semiconductor

Summary

The MIPI DSI/CSI-2 Transmitter IP receives MIPI D-PHY compatible byte stream, encapsulates it into DSI/CSI-2 data frames, and drives the Gowin MIPI D-PHY TX Advance IP interface to realize MIPI DSI/CSI-2 D-PHY transmission.
