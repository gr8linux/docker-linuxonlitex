DDRDLL

Information

Type: DDRDLL
Vendor: GOWIN Semiconductor
Summary:Gowin FPGA provides a Delay Locked Loop (DDRDLL), which provide code scaling capability to shift 90 degree delay code down to 45 degree or up to 135 degree by 11 degree jump.

Options

Lock & Code

  • Force Lock and Code - In this mode, the STEP value is forcibly set to 255.
  • Generated From DDRDLL Loop - This mode ensures STEP value is generated by the DDRDLL Loop.

Delay Code(degree)

  • Specify a delay code (degree) for DLL.

Lock Mode

  • Allows you to select Normal Lock Mode or Fast Lock Mode.
