DDRDLL

Information

Type: DDRDLL
Vendor: GOWIN Semiconductor

Summary

Gowin FPGA provides a Delay Locked Loop (DDRDLL), which provide code scaling capability to shift 90 degree delay code down to 45 degree or up to 135 degree by 11 degree jump.
