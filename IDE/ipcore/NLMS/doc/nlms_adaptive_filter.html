NLMS Adaptive Filter

Information

Type: NLMS Adaptive Filter
Vendor: GOWIN Semiconductor

Summary

The Gowin NLMS Adaptive Filter IP realizes the adaptive filtering function based on NLMS(Normalized Least Mean Squares) algorithm. It can be used for acoustic echo cancellation, etc. The structure of IP can be customized by the user including Number of Taps, Adaptation Step Size and Data Width.
