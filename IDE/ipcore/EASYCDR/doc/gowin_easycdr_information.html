EasyCDR

Information

Type: EasyCDR
Vendor: GOWIN Semiconductor

Summary

EasyCDR is a GPIO-based high-speed serial data receiving IP, no need to sampling-clock when receiving data. EasyCDR converts serial data into parallel data.
