MJPEG Decoder

Information

Type: MJPEG Decoder
Vendor: GOWIN Semiconductor

Summary

The Gowin MJPEG Decoder IP is a dynamic image frame-by-frame decoder that decompress and convert each frame of image data that has been compressed into Baseline JPEG standard into RGB format. Depending on the characteristics of the image and the ratio used for compression, the input and output rates are not fixed.

The design has these features:

  • Support input Baseline JPEG standard image and output RGB888
  • Support Pre-config image size include: 64x64 up to 1920x1080
  • Support dynamic sub-sampling of 4:4:4, 4:2:2, 4:2:0
  • Support dynamic Huffman table up to 2 DC and 2AC tables
  • Support dynamic Quantization table
