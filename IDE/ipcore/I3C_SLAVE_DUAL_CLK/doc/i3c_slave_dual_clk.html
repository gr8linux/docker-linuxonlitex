I3C Slave Dual Clock

Information

Type: I3C Slave Dual Clock
Vendor: GOWIN Semiconductor

Summary

The Gowin I3C Slave Dual Clock IP is a fully verified soft IP core, which can implement I3C bus communication. The IP implements the slave interface, which is defined in I3C V1.1 protocol.
