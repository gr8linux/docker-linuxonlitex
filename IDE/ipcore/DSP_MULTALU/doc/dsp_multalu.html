MULTALU

Information

Typ: MULTALU
Vendor: GOWIN Semiconductor

Summary

Each gwDSP macro can implement the Multiplier ALU mode. The MULTALU represents two functional modules, including MULTALU18x18 and MULTALU36x18.
The MULTALU18x18 can be configured in the following operation modes:

  • A*B+C
  • A*B-C
  • Accum+A*B+C
  • Accum+A*B-C
  • Accum-A*B+C
  • Accum-A*B-C
  • CASI+A*B
  • CASI+Accum+A*B
  • CASI+Accum-A*B
  • CASI+A*B+D
  • CASI+A*B-D

The MULTALU36x18 can be configured in the following operation modes:

  • A*B+C
  • A*B-C
  • Accum+A*B
  • CASI+A*B
