SDIO SDR104 Slave Controller

Information

Type: SDIO SDR104 Slave Controller
Vendor: GOWIN Semiconductor

Summary

SDIO SDR104 Slave Controller IP is a fully verified soft IP core, which can implement SDIO Slave Controller with SDR104 mode, and the IP can be controlled by SDIO Host. The IP implements Low-Speed, Full-Speed and UHS-I.
