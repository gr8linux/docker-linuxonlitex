SPI Nor Flash Interface(With Internal Flash)

Information

Type: SPI Nor Flash Interface(With Internal Flash)
Vendor: GOWIN Semiconductor

Summary

The Gowin SPI Nor Flash Interface(With Internal Flash) IP allows designers to create custom systems in one Gowin device that connect easily to Nor Flash chips. This IP located between the Nor Flash Memory and the user's logic, reduces the user's effort to deal with the Nor Flash Memory command interface by providing AHB interface to the user.
