DDR2 Memory Interface

Information

Type: DDR2 Memory Interface
Vendor: GOWIN Semiconductor

Summary

The Gowin DDR2 Memory Interface IP provides a complete solution for customers to use DDR2 Memory. This IP located between the DDR2 Memory and the user logic include with Gowin Memory Controller and Physical interface, reduces the user's effort to deal with the DDR2 Memory command interface by providing a simple generic system interface to the user.
