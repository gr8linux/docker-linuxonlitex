Adder Subtractor

Information

Type: Adder Subtractor
Vendor: GOWIN Semiconductor

Summary

The Gowin Adder Subtractor IP accept two integer as inputs, and produce two integer outputs, a carry out and a sum or difference. This IP supports signed and unsigned data representation for the input data.
