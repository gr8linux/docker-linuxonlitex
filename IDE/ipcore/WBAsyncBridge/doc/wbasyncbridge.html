WB Async Bridge

Information

Type: WB Async Bridge
Vendor: GOWIN Semiconductor

Summary

The Gowin WB Async Bridge IP is a bus bridge which can transform the wishbone master into an asynchronous wishbone slave. The Gowin WB Async Bridge IP can configure the Addr width as 8-64 and the Data width as 8,16,32,64.
