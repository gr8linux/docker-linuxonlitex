MIPI RX Advance

Information

Type: MIPI RX Advance
Vendor: GOWIN Semiconductor

Summary

The Mobile Industry Processor Interface (MIPI) is a specification standard used for interfaces to mobile application processors. MIPI D-PHY is a bus which provides a physical layer definition to implement high speed connectivity. The MIPI D-PHY is typically used for camera and display interfacing. The advanced FPGA can receive data which respects to the MIPI D-PHY specification through the Gowin MIPI D-PHY RX Advance IP. Furthermore, the IP can be used as the physical layer for higher level protocols such as the MIPI Camera Serial Interface (CSI-2) and Display Serial Interface (DSI).
