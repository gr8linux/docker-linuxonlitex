DDR3 Memory Interface

Note: If you want to use a higher version, please use Soft IP Core -> Memory Control -> DDRx SDRAM Memory Interface -> DDR3 Memory Interface

Information

Type: DDR3 Memory Interface
Vendor: GOWIN Semiconductor

Summary

The Gowin DDR3 Memory Interface IP provides a complete solution for customers to use DDR3 Memory. This IP located between the DDR3 Memory and the user logic include with Gowin Memory Controller and Physical interface, reduces the user's effort to deal with the DDR3 Memory command interface by providing a simple generic system interface to the user.
