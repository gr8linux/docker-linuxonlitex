DDR2 PHY Interface

Information

Type: DDR2 PHY Interface
Vendor: GOWIN Semiconductor

Summary

The Gowin DDR2 PHY Interface IP provides a complete solution for customers to use DDR2 Memory. This IP located between the DDR2 SDRAM Memory and the user's Memory Controller, reduces the user's effort to deal with the DDR2 SDRAM Memory with Gowin FPGA.
