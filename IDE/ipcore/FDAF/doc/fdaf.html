FD Adaptive Filter

Information

Type: FD Adaptive Filter
Vendor: GOWIN Semiconductor

Summary

The Gowin FD Adaptive Filter(Frequency domain adaptive filter) IP implements an adaptive finite impulse response (FIR) filter in the frequency domain using the fast block least mean square (LMS) algorithm.
