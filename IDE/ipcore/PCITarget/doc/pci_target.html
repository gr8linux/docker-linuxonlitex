PCI Target

Information

Type: PCI Target
Vendor: GOWIN Semiconductor

Summary

The Gowin PCI Target Local Bus is a high performance 32bit bus with multiplexed address and data lines. The bus is intended for use as an interconnect mechanism between highly integrated peripheral controller components, peripheral add-in boards, and processor/memory systems.
