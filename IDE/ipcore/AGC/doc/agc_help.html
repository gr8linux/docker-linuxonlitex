AGC

Information

Type: AGC
Vendor: GOWIN Semiconductor
Summary: Gowin AGC(Automatic Gain Control) IP is used to exhibit slow changes of the gain factor. During speech intervals the short-term power of the input signal is estimated and compared to a desired value. If the speech level exceeds an upper bound, the gain factor g(n) is decreased, in the other case the gain is increased. In order to avoid fast gain variations during short speech pauses g(n)is incremented or decremented in very small steps.

Options & Description

Gain

GINC :

  • Specify increased gain factor.

GDEC :

  • Specify decreased gain factor.

Threshold

KN :

  • Specify the threshold of noise level.

KP :

  • Specify the threshold of desired speech level.

Filter Constants

RSR :

  • Specify the rising constant of first slow smoothing filter.

RSF :

  • Specify the falling constant of first slow smoothing filter.

RFR :

  • Specify the rising constant of fast smoothing filter.

RFF :

  • Specify the falling constant of fast smoothing filter.

RPR :

  • Specify the rising constant of second slow smoothing filter.

RPF :

  • Specify the falling constant of second slow smoothing filter.
