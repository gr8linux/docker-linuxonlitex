UHS2 PSRAM Memory Interface

Information

Type: UHS2 PSRAM Memory Interface
Vendor: GOWIN Semiconductor

Summary

This PSRAM memory interface controller is designed to connect the embedded AP26408A-OKX PSRAM. PSRAM is embeded in follow Gowin FPGA:
GW5AR-LV25UG256P(AP26408A-OKX, capacity: 128Mb)
GW5ART-LV15CM90P(AP26408A-OKX, capacity: 128Mb)
GW5ART-LV15CM90PF(AP26408A-OKX, capacity: 128Mb)
GW5ART-LV15CS126P(AP26408A-OKX, capacity: 128Mb)
GW5ART-LV15MG132P(AP26408A-OKX, capacity: 64Mb)
GW5ANRT-LV15MG132P(AP26408A-OKX, capacity: 64Mb)
