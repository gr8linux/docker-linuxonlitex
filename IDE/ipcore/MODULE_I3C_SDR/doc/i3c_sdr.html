I3C SDR

Information

Type: I3C SDR
Vendor: GOWIN Semiconductor

Summary

Gowin I3C IP supports high-speed, low-power, and other critical features that are currently covered by I2C and SPI. The structure of the IP provides registers that enable users to control and implement specific functions flexibly.
