FIFO HS

Information

Type: FIFO HS
Vendor: GOWIN Semiconductor

Summary

FIFO HS IP is a dual clock input FIFO with independent read and write clocks. The difference between “FIFO” and “FIFO HS” is that “FIFO HS” uses more logic resources, but has faster performance. Use “FIFO” for resource limited designs and “FIFO HS” for performance optimized designs for optimized results.
