USB to Multi Serial Protocol Bridge

Information

Type: USB to Multi Serial Protocol Bridge
Vendor: GOWIN Semiconductor

Summary

The Gowin USB to Multi Serial Protocol Bridge IP provides UART, JTAG, SPI, I2C interface for Gowin FPGA to receive and send data.

The design has these features:
● Simultaneously supports four interfaces: uart, jtag, spi, and i2c.
● Interface parameters can be independently configured.
