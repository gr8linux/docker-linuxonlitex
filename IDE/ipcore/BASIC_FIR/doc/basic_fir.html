Basic FIR Filter

Information

Type: Basic FIR Filter
Vendor: GOWIN Semiconductor

Summary

The Gowin Basic FIR Filter IP is a basic version of FIR filter. It realizes FIR filter function by completing multiplication and accumulation operation. Mult Array Length determines the number of internal parallel multipliers, and the larger the number, the faster the calculation speed.
