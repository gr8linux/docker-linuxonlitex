RAM16S

Information

Type: RAM16S
Vendor: GOWIN Semiconductor

Summary

Shadow memory can be configured as single port mode. In this mode, the shadow memory supports different read and write data widths, and it can be implemented by primitive RAM16S1, RAM16S2 and RAM16S4 or a combination thereof. It can also be initialized by a memory initialization file.
