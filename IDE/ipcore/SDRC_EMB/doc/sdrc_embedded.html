SDRAM Controller(With embedded SDRAM)

Information

Type: SDRAM Controller
Vendor: GOWIN Semiconductor

Summary

This controller is dedicated for embedded SDRAMs in the following devices. It should be considered for legacy designs. New designs should consider using SDRAM Controller HS.

  • 1NR-4-QN88 (sipped 64Mbit SDRAM)
  • 1NR-9-QN88 (sipped 64Mbit SDRAM)
  • 2ANR18-QN88 (sipped 64Mbit SDRAM & 32Mbit FLASH)
  • 2AR18-EQ144/QN88/PG256S(sipped 64Mbit SDRAM)
