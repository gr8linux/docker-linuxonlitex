FIFO

Information

Type: FIFO
Vendor: GOWIN Semiconductor

Summary

FIFO IP is a dual clock input FIFO with independent read and write clocks. The difference between “FIFO” and “FIFO HS” is that “FIFO” uses less logic resources, but has slower performance as a result. Use “FIFO” for resource limited designs and “FIFO HS” for performance optimized designs for optimized results.
