BCH Encoder

Information

Type: BCH Encoder
Vendor: GOWIN Semiconductor

Summary

The Gowin BCH Encoder IP is a Encoder based on BCH coding. The IP can be used for appending the check symbols to the encoded input message to form the output.

The design has these features:

  • 8-32 bit input/output data width
  • 4-16 Galois field
