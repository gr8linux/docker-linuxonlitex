MULTADDALU

Information

Type: MULTADDALU
Vendor: GOWIN Semiconductor

Summary

Each gwDSP macro can implement the sum of two 18x18 multipliers. The alu two-18x18 mode can be composed of two 18x18 multipliers and one ALU. The MULTADDALU can be configured to work in the following operation modes:

  • A0*B0 + A1*B1
  • A0*B0 - A1*B1
  • A0*B0 + A1*B1 + C
  • A0*B0 + A1*B1 - C
  • A0*B0 - A1*B1 + C
  • A0*B0 - A1*B1 - C
  • Accum + A0*B0 + A1*B1
  • Accum + A0*B0 - A1*B1
  • A0*B0 + A1*B1 + CASI
  • A0*B0 - A1*B1 + CASI
