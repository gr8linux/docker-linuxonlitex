BandGap

Information

Type: BandGap
Vendor: GOWIN Semiconductor

Summary

In the supported device, the function of bandGap is to provide constant voltage and current for some modules in the chip. If bandGap is turned off, some modules like OSC, PLL and FLASH will not work, thus reducing device power consumption.
