SerDes

Information

Type: SerDes
Vendor: GOWIN Semiconductor

Summary

The Gowin SerDes IP is a comprehensive IP base on Gowin SerDes. It includes various high speed protocol, such as Serial Ethernet, CPRI, JESD204B, PCIe, etc. It also provides a Customized PHY to configure Gowin SerDes flexibly.

The list of protocols is as follows:

  • BCDR
  • 1G Serial Ethernet
  • 2.5G Serial Ethernet
  • 10G Serial Ethernet
  • QSGMII
  • RoraLink 64B66B
  • RoraLink 8B10B
  • CPRI
  • JESD204B
  • Serial RapidIO
  • USB 3.0 PHY
  • EDP PHY
  • SDI
  • SDI PHY
  • SLVS-EC RX
  • Customized PHY
  • PCI Express Controller
