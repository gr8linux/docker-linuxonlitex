RoraLink 8B10B

Information

Type: RoraLink 8B10B
Vendor: GOWIN Semiconductor

Summary

The Gowin RoraLink 8B10B IP is a scalable, lightweight, link-layer protocol for high-speed serial communication. The IP is typically used in applications requiring simple, low-cost, high-rate, data channels and is used to transfer data between devices using one or more transceivers.
