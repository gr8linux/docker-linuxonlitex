1G Serial Ethernet

Information

Type: 1G Serial Ethernet
Vendor: GOWIN Semiconductor

Summary

The Gowin 1G Serial Ethernet IP provides customers with a 1G Serial Ethernet solution that supports the 1000BASE-X protocol in the IEEE802.3 Standard and the SGMII protocol and implements Ethernet MAC layer function.
