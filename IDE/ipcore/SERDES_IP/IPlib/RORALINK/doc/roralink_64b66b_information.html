RoraLink 64B66B

Information

Type: RoraLink 64B66B
Vendor: GOWIN Semiconductor

Summary

The Gowin RoraLink 64B66B IP is a scalable, lightweight, high data rate link-layer protocol for high-speed serial communication. The cores can be simplex or full-duplex, featuring one of two simple user interfaces, and optional flow control.
