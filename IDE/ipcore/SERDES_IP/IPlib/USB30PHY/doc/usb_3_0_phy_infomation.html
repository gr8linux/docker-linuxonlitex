USB 3.0 PHY

Information

Type: USB 3.0 PHY
Vendor: GOWIN Semiconductor

Summary

The USB 3.0 PHY IP core supports 5.0-Gbps USB 3.0 physical layer transceiver. It support the 32-bit PIPE operates with a 125-MHz interface clock.
