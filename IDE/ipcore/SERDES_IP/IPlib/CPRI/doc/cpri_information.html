CPRI

Information

Type: CPRI
Vendor: GOWIN Semiconductor

Summary

Gowin CPRI implements the CPRI Specification V7.0(2015-10-09). CPRI is a high-speed serial interface for network radio equipment controllers(REC) to receive data from and provide data to remote radio equipment(RE).
