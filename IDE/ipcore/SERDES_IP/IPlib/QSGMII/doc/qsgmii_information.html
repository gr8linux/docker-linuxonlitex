QSGMII

Information

Type: QSGMII
Vendor: GOWIN Semiconductor

Summary

The Gowin QSGMII IP provides customers with a SerDes-based solution that combines four SGMIIs into one 5 Gigabits per second (Gb/s) interface. It supports the QSGMII protocol and implements Ethernet MAC layer function.
