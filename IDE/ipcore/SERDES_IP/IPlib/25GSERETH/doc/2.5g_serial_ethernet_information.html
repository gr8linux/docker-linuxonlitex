2.5G Serial Ethernet

Information

Type: 2.5G Serial Ethernet
Vendor: GOWIN Semiconductor

Summary

The Gowin 2.5G Serial Ethernet IP provides customers with a 2.5G Serial Ethernet solution that supports the 2500BASE-X protocol and the 2.5G SGMII protocol and implements Ethernet MAC layer function.
