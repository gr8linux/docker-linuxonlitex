PCI Express Controller

Information

Type: PCI Express Controller
Vendor: GOWIN Semiconductor

Summary

Gowin PCI Express Controller core is a customizable building block of GW5AT FPGA devices. It is compliant with PCI Express Base Specification Rev 3.0, supporting Endpoint configuration with x1, x2, x4 mode at up to 8Gbps (PCI Express Gen3).
