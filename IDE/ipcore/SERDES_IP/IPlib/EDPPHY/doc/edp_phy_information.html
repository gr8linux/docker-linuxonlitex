Information

Type: EDP PHY
Vendor: GOWIN Semiconductor

Summary

DisplayPort is a digital video interface standard developed by the PC and chip manufacturer alliance. EDP (Embedded DisplayPort) is an extension of the embedded architecture and protocol for DP (DisplayPort) applications, so the eDP protocol is fully compatible with the DP protocol. EDP PHY is a bus that provides a physical layer definition to implement high-speed connectivity. The EDP PHY is typically used for display interfacing.
