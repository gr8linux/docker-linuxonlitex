SLVS-EC RX

Information

Type: SLVS-EC RX
Vendor: GOWIN Semiconductor

Summary

The Gowin SLVS-EC RX IP provides SLVS-EC interface for Gowin FPGA to receive image sensor data.

The design has these features:
● Compliant with SLVS-EC Specification Version 1.2
● Supports Error Correction Code(ECC) or Cyclic Redundancy Check(CRC)
● Supports Byte-to-Pixel conversion for various lane-configurations
● Supports various functions defined by the SLVS-EC Link layer
