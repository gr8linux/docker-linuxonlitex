Color_Space_Convertor

Information

Type: Color_Space_Convertor
Vendor: GOWIN Semiconductor

Summary

Color Space Convertors are used to convert signals from one color space to another color space, which can be used in video and image display systems. The Color Space Convertor supports a number of predefined conversions between standard color spaces, and allows the entry of custom coefficients to translate between any two three-valued color spaces.
