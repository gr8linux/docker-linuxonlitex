USB Audio Class

Information

Type: USB Audio Class
Vendor: GOWIN Semiconductor

Summary

Gowin USB Audio Class IP supports high speed and full speed modes. It supports up to 3 input and 3 output audio devices, and each audio device can be configured to be in synchronous mode or asynchronous mode. It can select the bit depth (16bit/24bit/32bit) and the sampling frequency (32kHz/48kHz/96kHz/192kHz/384kHz/768kHz) supported by the device. It can be connected to audio devices that support IIS/PCM protocols and PCM data format.
