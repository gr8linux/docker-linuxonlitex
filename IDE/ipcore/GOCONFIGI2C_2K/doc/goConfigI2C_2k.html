goConfig I2C

Information

Type: goConfig I2C
Vendor: GOWIN Semiconductor

Summary

The goConfig I2C IP can realize the online upgrade function of the I2C port. It can burn the FPGA through the I2C port to replace the internally stored data and realize background burning.
