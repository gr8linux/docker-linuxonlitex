Revised DP

Information

Type: Revised DP
Vendor: GOWIN Semiconductor

Summary

This revised Dual Port BSRAM uses 16/18 data width to implement low-width configuration, and will increase design resources when memory exceeds 16K/18K.
BSRAM can be configured as Dual Port mode. In this mode, Port A and Port B can read/write independently.
