10G Ethernet MAC

Information

Type: 10G Ethernet MAC
Vendor: GOWIN Semiconductor

Summary

The Gowin 10G Ethernet MAC IP provide customers with 10G Ethernet MAC, compliant with IEEE802.3 clause 46. The IP supports XGMII interface and multiple configurations for applications.
