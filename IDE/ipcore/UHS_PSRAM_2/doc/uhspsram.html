UHS PSRAM Memory Interface V2.0

Information

Type: UHS PSRAM Memory Interface V2.0
Vendor: GOWIN Semiconductor

Summary

This PSRAM Memory Interface Controller is for interfacing devices with embedded or external AP Memory PSRAM. This controller interfaces to one 8-bit PSRAM memory. V1.0 has PLL delay auto calibration within the IP. V2.0 has a fixed delay value that is configured while generating the IP with a default value optimized for embedded devices. The IP supports APS3208k-OK, PSRAM is included in the following FPGA:
GW1NR-LV9MG100PA(APS3208k-OK, capacity: 128Mb)
GW1NR-LV2MG49P, GW1NR-UV2MG49P(APS3208k-OK, capacity: 64Mb)
GW1NR-LV2MG49PG, GW1NR-UV2MG49PG(APS3208k-OK, capacity: 32Mb)
