Reed-Solomon Decoder

Information

Type: Reed-Solomon Decoder
Vendor: GOWIN Semiconductor

Summary

The Gowin Reed-Solomon Decoder IP is used in many Forward Error Correction applications and communications systems. In an error correction system, the decoder receives data and corrects error caused by transmission.
