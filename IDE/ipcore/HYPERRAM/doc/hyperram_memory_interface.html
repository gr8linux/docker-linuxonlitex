HyperRAM Memory Interface

Information

Type: HyperRAM Memory Interface
Vendor: GOWIN Semiconductor

Summary

This HyperRAM Memory Interface Controller is for interfacing with Cypress PSRAM. The IP supports W956D8MKY-5I, S27KS0641, IS66WVH32M8DBLL, etc. There are two Gowin FPGA chips embedded with W956D8MKY-5I, They are GW1NSR-LV4CQN48P(capacity: 64Mb) and GW1NSER-LV4CQN48P(capacity: 64Mb)
