PDM2PCM

Information

Type: PDM2PCM
Vendor: GOWIN Semiconductor

Summary

Gowin PDM2PCM IP is a design to convert PDM data to PCM data. Partial configuration of the IP can be customized by the user including Input channels, Edge mode, Rate, Stages, Differential Delay and so on.
