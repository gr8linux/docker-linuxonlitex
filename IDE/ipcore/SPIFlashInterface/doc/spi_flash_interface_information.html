SPI Flash Interface(With External Flash)

Information

Type: SPI Flash Interface(With External Flash)
Vendor: GOWIN Semiconductor

Summary

The Gowin SPI Flash Interface(With External Flash) IP allows designers to create custom systems in one Gowin device that connect easily to external SPI Flash chips. This IP located between the SPI Flash Memory and the user's logic, reduces the user's effort to deal with the SPI Flash Memory command interface by providing AHB and APB interface to the user.
