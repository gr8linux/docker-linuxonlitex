CLKDIVG

Information

Type: CLKDIVG
Vendor: GOWIN Semiconductor
Summary: Gowin FPGA provides a clock division module (CLKDIVG), which is used for clock division. The division factors supported by GW2AN-18X include 2, 3.5, 4, 5, other devices include 2, 3.5, 4, 5 and 8. The function of CLKDIVG is the same as CLKDIV, but the position is different.
Devices: GW1N-2, GW1NR-2, GW2AN-18X, GW1N-1P5

Options

Division Factor

  • Allows you to select a division factor.

Calibration

  • Selecting this option will produces the calib port in the generated module, which is used to calibrate the clock signal.
