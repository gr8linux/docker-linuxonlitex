CLKDIVG

Information

Type: CLKDIVG
Vendor: GOWIN Semiconductor

Summary

Gowin FPGA provides a clock division module (CLKDIVG), which is used for clock division. The function of CLKDIVG is the same as CLKDIV, while the position is different.
