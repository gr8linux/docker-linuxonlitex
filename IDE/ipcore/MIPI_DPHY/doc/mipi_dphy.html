MIPI_DPHY

Information

Type: MIPI_DPHY
Vendor: GOWIN Semiconductor

Summary

Gowin IPCore MIPI_DPHY can implement register access interface for DPHY working mode configuration, and support DPHY HS-TX and HS-RX.
