SDRAM Controller HS

Information

Type: SDRAM Controller HS
Vendor: GOWIN Semiconductor
Summary: The SDRAM controller HS with high read and write efficiency allows designers to create custom systems in one Gowin device that connect easily to SDRAM chips. This SDRAM Controller located between the SDRAM and the user logic, reduces the user's effort to deal with the SDRAM command interface by providing a simple generic system interface to the user.
