NPU

Information

Type: NPU
Vendor: GOWIN Semiconductor

Summary

NPU is Neural-network Processing Unit to accelerate machine learning computing.

NPU contains four functional units, AHB bridge unit, accelerator unit, PSRAM controller unit and SPI flash controller unit.

AHB bridge unit, is to connect to a MCU, wok with this MCU together.

Accelerator unit is a set of machine learning algorithms, modeled loosely after the human brain, that are designed to recognize patterns. They interpret sensory data through a kind of machine perception, labeling or clustering raw input. This accelerator unit contains DepthwiseConv2D, Conv2D, MaxPool2D and AveragePool2D.

PSRAM controller unit is to save some temporary data for accelerator unit. When accelerator unit is computing each model layer, PSRAM reads data of previous layer data, and writes data of current layer to next layer.

SPI flash controller unit is to save weight data and bias data for accelerator unit. When accelerator unit is computing each model layer, will read weight data and bias data of this layer from SPI flash.
