FIFO SC HS

Information

Type: FIFO SC HS
Vendor: GOWIN Semiconductor
Summary: The Gowin FIFO SC HS IP is a fully verified, synchronous first-in first-out memory queue. The structure of the IP can be customized by the user including the write depth, the write width, memory type, almost empty and full flags and so on.

Options & Description

FIFO Implementation :

  • The memory type used to implement the FIFO SC HS, including BSRAM(block ram), SSRAM(distributed ram) and REG(registers).

Write Depth :

  • Specify the number of the address locations for input.

Write Data Width :

  • Specify the data width of the input data.

Read Depth :

  • The number of the address locations for output, which is same with the Write Depth.

Read Data Width :

  • The data width for the output data, which is same with the Write Data Width.

Write Data Num :

  • This enables the data count output, which is synchronized with clock and shows the number of word in the FIFO SC HS.

Almost Full Flag :

  • Enable the almost full flag.

Full-Single Threshold Constant Parameter :

  • A single threshold is set at the time of the IP generation.

Full-Dual Threshold Constant Parameters :

  • Dual thresholds are set at the time of the IP generation.

Full-Single Threshold Input Parameter :

  • A single threshold is input from the additional port to allow changing the threshold during operation.

Full-Dual Threshold Input Parameters :

  • Dual thresholds are input from the additional ports to allow changing the thresholds during operation.

Set :

  • The threshold of setting the almost full flag to 1.

Clear :

  • The threshold of clearing the almost empty flag to 0.

Almost Empty Flag :

  • Enable the almost empty flag.

Empty-Single Threshold Constant Parameter :

  • A single threshold is set at the time of the IP generation.

Empty-Dual Threshold Constant Parameters :

  • Dual thresholds are set at the time of the IP generation.

Empty-Single Threshold Input Parameter :

  • A single threshold is input from the additional port to allow changing the threshold during operation.

Empty-Dual Threshold Input Parameter :

  • Dual thresholds are input from the additional ports to allow changing the thresholds during operation.

Set :

  • The threshold of setting the almost empty flag to 1.

Clear :

  • The threshold of clearing the almost empty flag to 0.

ECC Selected :

  • Allow error correction of single error and error detection of double errors. This is not supported for data widths greater than 64 bits, and only supported for block ram.

Output Register Selected :

  • Registers are used for the output data, which can't aspect the flag signal.

Controlled by RdEn :

  • The Registers for the output data are controlled by RdEn, and the last data can't be output.
