OSC

Information

Type: OSC
Vendor: GOWIN Semiconductor
Summary: Internal clock generator OSC. The user can change the value of the option "Frequency Divider" to generate a different output clock frequency, which can be one of 64 values: 1/2, 1/4, 1/6, 1/8, 1/10, ... or 1/128 of the oscillator frequency. Different devices have different oscillator frequencies, GW1N-4/GW1N-4B/GW1N-4D/GW1NR-4/GW1NR-4B/GW1NR-4D/GW1NRF-4B/GW1N-9/GW1N-9C/GW1NR-9/GW1NR-9C is about 210MHz, GW1N-1/GW1NR-1/GW1N-1S/GW1NS-2/GW1NS-2C/GW1NSR-2/GW1NSR-2C/GW1NSE-2C is about 240MHz, GW2AN-18X is about 200MHz, other devices are about 250MHz.
Devices: GW1N-1, GW1NR-1, GW1NZ-1, GW1NZ-1C, GW1N-1S, GW1N-2, GW1NR-2, GW1N-4, GW1N-4B, GW1NR-4, GW1NR-4B, GW1NR-4D, GW1NRF-4B, GW1N-4D, GW1NS-2, GW1NS-2C, GW1NSR-2, GW1NSR-2C, GW1NSE-2C, GW1NS-4, GW1NS-4C, GW1NSR-4, GW1NSER-4C, GW1NSR-4C, GW1N-9, GW1N-9C, GW1NR-9, GW1NR-9C, GW2A-18, GW2AR-18, GW2A-18C, GW2AR-18C, GW2ANR-18C, GW2A-55, GW2A-55C, GW2AN-55C, GW2AN-18X, GW1N-1P5

Options

Frequency Divider

  • Allows you to select any even number between 2 ~ 128.

Regulator Enable

  • If the device is GW1N-2/GW1NR-2/GW1N-1P5, when this option is selected, OSC will be powered by the regulator.
