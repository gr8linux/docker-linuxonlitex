Fixed Point Divider

Information

Type: Fixed Point Divider
Vendor: GOWIN Semiconductor

Summary

The Gowin Fixed Point Divider IP is a fully verified soft IP core, which can implement integer, fractional, and negative division. Partial configuration of the IP can be customized by the user including data width, fractional precision and so on.
