RAM16SDP

Information

Type: RAM16SDP
Vendor: GOWIN Semiconductor

Summary

Shadow memory can be configured as semi dual port mode. In this mode, the shadow memory supports different read and write data widths, and it can be implemented by primitive RAM16SDP1, RAM16SDP2 and RAM16SDP4 or a combination thereof. It can also be initialized by a memory initialization file.
