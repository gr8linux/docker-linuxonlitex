PLL_ADV

Information

Type: PLL_ADV
Vendor: GOWIN Semiconductor

Summary

Gowin FPGA provides a Phase Locked Loop (PLL), which is used to generate multiple clocks with defined phase and frequency relationships to a given input clock.
