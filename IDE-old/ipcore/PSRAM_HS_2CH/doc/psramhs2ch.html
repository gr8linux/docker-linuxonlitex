PSRAM Memory Interface HS 2CH

Information

Type: PSRAM Memory Interface High Speed 2CH
Vendor: GOWIN Semiconductor

Summary

This PSRAM Memory Interface Controller is for interfacing devices with embedded Winbond PSRAM. This controller interfaces to two 8-bit PSRAM memories (16-bit bus total). V1.0 has PLL delay auto calibration within the IP. V2.0 has a fixed delay value that is configured while generating the IP with a default value optimized for embedded devices.
