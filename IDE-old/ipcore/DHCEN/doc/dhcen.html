DHCEN

Information

Type: DHCEN
Vendor: GOWIN Semiconductor

Summary

Gowin FPGA provides a dynamic high-speed clock enable block (DHCEN), which is used for dynamically enable or disable the high-speed clock signal. Clock enable signal is active low.
