SDRAM Controller(With external SDRAM)

Information

Type: SDRAM Controller
Vendor: GOWIN Semiconductor

Summary

SDRAM Controller for any GOWIN device interfacing with external SDRAM. It should be considered for legacy designs. New designs should consider using SDRAM Controller HS.
