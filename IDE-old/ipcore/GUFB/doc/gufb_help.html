User Flash

Information

Type: User Flash
Vendor: GOWIN Semiconductor
Summary: GW1N series FPGA products provide user Flash memory resources (User Flash). Different devices support different Flash, including FLASH96K, FLASH64K, FLASH64KZ, FLASH128K, FLASH256K, FLASH256KA and FLASH608K.
Devices: GW1N-1, GW1NR-1, GW1NZ-1, GW1NZ-1C, GW1N-1S, GW1N-2, GW1NR-2, GW1N-4, GW1N-4B, GW1NR-4, GW1NR-4B, GW1NR-4D, GW1NRF-4B, GW1N-4D, GW1NS-2, GW1NSR-2 GW1NS-4, GW1NSR-4, GW1N-9, GW1N-9C, GW1NR-9, GW1NR-9C, GW1N-1P5

Note

  • If the target device is GW1N-1/GW1NR-1/GW1N-1S, the primitive FLASH96K will be instantiated in the customized module. For the primitive FLASH96K, data input and data output’s width is 32, input RA and CA’s width is 6.
  • If the target device is GW1NZ-1/GW1NZ-1C, the primitive FLASH64KZ or FLASH64K will be instantiated in the customized module. For the primitive FLASH64KZ or FLASH64K, data input and data output’s width is 32, input XADR’s width is 5 and YADR’s width is 6. Compared with FLASH64KZ, FLASH64K has a SLEEP port.
  • If the target device is GW1NS-2/GW1NSR-2, the primitive FLASH128K (128K Bytes) will be instantiated in the customized module. For the primitive FLASH128K, data input and data output’s width is 32, input ADDR’s width is 15.
  • If the target device is GW1N-4/GW1N-4B/GW1N-4D/GW1NS-4/GW1NSR-4/GW1NR-4/GW1NR-4B/GW1NR-4D/GW1NRF-4B, the primitive FLASH256K will be instantiated in the customized module. For the primitive FLASH256K, data input and data output’s width is 32, input XADR’s width is 7, input YADR’s width is 6.
  • If the target device is GW1N-2/GW1NR-2/GW1N-1P5, the primitive FLASH256KA will be instantiated in the customized module. For the primitive FLASH256KA, data input and data output’s width is 32, input XADR’s width is 7, input YADR’s width is 6. FLASH256KA has a sleep signal.
  • If the target device is GW1N-9/GW1N-9C/GW1NR-9/GW1NR-9C, the primitive FLASH608K will be instantiated in the customized module. For the primitive FLASH608K, data input and data output’s width is 32, input XADR’s width is 9, input YADR’s width is 6.
