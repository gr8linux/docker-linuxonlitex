User Flash

Information

Type: User Flash
Vendor: GOWIN Semiconductor

Summary

GW1N series FPGA products provide user Flash memory resources (User Flash). Different devices support different Flash, including FLASH96K, FLASH64K, FLASH64KZ, FLASH128K, FLASH256K, FLASH96KA and FLASH608K.
