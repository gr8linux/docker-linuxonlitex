Gowin Flash Controller

Information

Type: Gowin Flash Controller
Vendor: GOWIN Semiconductor

Summary

GW1N series FPGA products provide user Flash memory resources (User Flash). Different devices support different Flash, including FLASH96K, FLASH96KA, FLASH64KZ, FLASH64K, FLASH128K, FLASH256K and FLASH608K.

Gowin Flash Controller is a fully verified soft IP core which can control User Flash to read and write data. Controller includes three interfaces, Register interface, AHB Bus interface and Wishbone Bus interface. Wishbone Bus interface is to connect Wishbone Bus in MCU with Wishbone Bus, and AHB Bus interface is to connect AHB Bus in MCU with AHB Bus, as an instruction memory or data storage for MCU, as an instruction memory or data storage for MCU.
