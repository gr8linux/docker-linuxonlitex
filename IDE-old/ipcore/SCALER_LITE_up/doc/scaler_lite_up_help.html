Scaler Lite Up

Information

Type: Scaler Lite Up
Vendor: GOWIN Semiconductor
Summary: The Scaler Lite Up IP is used to convert the input video frames from one size up to another size. It only supports bilinear algorithm and 1:2, 2:3, 3:4 zoom ratio.

Options & Description

Data Setting

Data Format :

  • The format of input data.

Zoom Ratio :

  • The zoom ratio of scaler.

Input Frame Width :

  • The horizontal resolution of input frame.

Input Frame Height :

  • The vertical resolution of input frame.

Fetch Data Delay Value :

  • The data delay value when read data from video frame buffer.
